# hn
A Hacker News reader iOS app written in Swift.

# Used
- MVP architecture.
- Hacker News API.
- HTTP requests with URLSession.
- Without the use of storyboards.
- Support for light and dark mode.
- Use UserDefaults for saving theme.
- Unit Tests
